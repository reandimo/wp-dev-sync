Edit locally. Auto-sync on save. SSH or FTP.
Like shopify theme dev, but for WordPress.

- The Old Way

  • Open FileZilla
  • Navigate to remote dir
  • Upload files manually
  • Refresh browser
  • Repeat 500Г— a day

+ The wp-dev-sync Way

  • Run wp-dev-sync watch
  • Edit files in your editor
  • Files sync automatically
  • SSH delta = only changed bytes
  • Done. Go build things.

Think shopify theme dev, but for WordPress.

SSH (rsync) RECOMMENDED
Delta transfer вҖ” only changed bytes
  • Fast delta sync (~200 bytes per edit)
  • Encrypted connection
  • Passwordless with SSH keys
  • Checksum-based change detection
Install: choco install rsync В· brew install rsync
FTP (lftp) FALLBACK
Full file mirror вҖ” works everywhere
  • Universal hosting support
  • No server setup needed
  • Works with any cPanel/Plesk
  • Transfers full files, not deltas
Install: choco install lftp В· brew install lftp
